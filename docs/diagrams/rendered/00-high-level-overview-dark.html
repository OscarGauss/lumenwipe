LumenWipe — How it works

Non-custodial recovery of locked Stellar reserves. Keys never leave the browser.

Non-custodial by construction

  • • Keys never leave the user's browser
  • • The backend never sees a private key
  • • User reviews every transaction
  • • Open-source, replaceable, self-hostable

Read-only data sources

  • • Stellar RPC: simulate, submit, events
  • • stellar.expert: enumerate subentries
  • • OctoPos / Orion: DeFi positions
  • • Soroswap: route quotes

Stellar network

  • • Classic + Soroban operations
  • • User-signed XDR submitted directly
  • • Recovered XLM goes to the destination
  • • Single source of truth at every step

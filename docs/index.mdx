LumenWipe is an open-source, non-custodial tool that closes a Stellar account from start to finish. It unwinds trustlines, offers, data entries, extra signers, liquidity pool shares, and DeFi positions, converts what is left to XLM, and merges the account into a destination wallet or exchange. Every transaction is signed in the browser, so no server ever holds a key or moves funds.

It extends the public-domain [stellar.expert/demolisher](https://stellar.expert/demolisher/public) by Orbit Lens with full Soroban and DeFi support.
